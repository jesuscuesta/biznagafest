IFRAME

Con un iframe podremos referenciar cualquier web, para que se pueda visualizar dentro de este tag. Una de las limitaciones que tiene, es que para que se pueda visualizar, deberemos tener configurado el CORS para poder ver la web, dentro de nuestro dominio.

Más información en página de mozilla: https://developer.mozilla.org/es/docs/Web/HTML/Element/iframe
